Apache FOP Servlet

Purpose

This servlet created to work with APEX Reports and produce PDF documents from XML and XSLT templates. Servlet accepts two parameters.
Parameter name Parameter description
xml Data in XML format to be printed
template XSLT template to generate XSL-FO document
GET Call /pdf_print.jsp?xml=<...>&template=<...>

Parameter values are URL-encoded data.
