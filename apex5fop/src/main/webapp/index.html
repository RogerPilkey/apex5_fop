Apache FOP Servlet


This servlet has been developed as the replacement for the original Oracle FOP servlet for Apex reports.
It could be configured with Oracle Apex 5 Reports and to generate PDF documents from the XML data and XSLT templates.
Servlet accepts two parameters. as in the table below.

Parameter name Parameter description
xml Data in XML format to be printed
template XSLT template to generate XSL-FO document
Servlet requests
GET
(Values are URL-encoded)		 /pdf_print.jsp?xml=…&template=…
